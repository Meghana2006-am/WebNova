🛡️ Raksha Saheli – Voice Safety Companion

🌟 Overview

Raksha Saheli is an intelligent, voice-enabled safety web application designed to provide instant emergency assistance during critical situations. It supports hands-free interaction, making it highly useful when users are unable to operate their device manually.

The system combines SOS alerts, voice commands, real-time location tracking, and emergency contact management into a single, user-friendly platform.

---

🚀 Key Features

🎙️ Voice Command System (Highlight Feature)

- Activate emergency actions using voice
- Supports commands like:
  - “SOS”
  - “Call [Contact Name]”
  - “Share location”
  - “Find police”
- Enables hands-free safety support

---

🚨 Smart SOS Emergency System

- One-tap SOS activation
- Triggers:
  - Alarm sound 🔊
  - Device vibration 📳
  - Automatic emergency call 📞
- Displays full-screen emergency alert

---

👥 Emergency Contact Management

- Add custom emergency contacts
- Call contacts instantly
- Delete or update contacts
- Stored using local storage for quick access

---

📍 Live Location Sharing

- Fetches real-time GPS location
- Shares Google Maps link via WhatsApp
- Helps responders locate user quickly

---

🚔 Nearby Police Detection

- Uses OpenStreetMap API
- Displays nearby police stations
- Direct navigation via Google Maps

---

🌐 Universal SOS Sharing

- Generate and share SOS alert link instantly
- Can be sent to multiple people

---
